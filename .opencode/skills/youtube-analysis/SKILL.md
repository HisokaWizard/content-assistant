# Skill: youtube-analysis

## Назначение

Используй этот skill, когда пользователь просит анализ YouTube-видео по URL.

## Вход

- `youtubeUrl`: полная ссылка на YouTube-видео.
- `criteria` (optional): критерии оценки от пользователя.
- `interests` (optional): интересы пользователя.

## Алгоритм

1. Получи metadata видео.
2. Получи transcript бесплатным способом (без YouTube API ключа).
3. Определи язык transcript.
4. Если язык не русский, переведи итоговый summary/тезисы на русский.
5. Сгенерируй структурированный ответ.

## Transcript-First

- Анализ разрешен только если transcript успешно получен.
- Если transcript отсутствует или слишком короткий, верни отказ: "анализ без transcript невозможен".
- Нельзя делать анализ только по заголовку/описанию/метаданным.

## Формат ответа

```text
1) Коротко о видео (2-3 предложения): тема, цель, кому полезно.
2) Главные тезисы (6-10 пунктов).
3) Факты и аргументы (5-8 пунктов): цифры, примеры, причинно-следственные связи.
4) Позиция автора(ов):
   - ключевые идеи;
   - на чем они основаны;
   - уязвимые места аргументации.
5) Выводы и результаты для зрителя.
6) Практические шаги (3-5 действий).
7) Релевантность интересам пользователя: 0-10 + краткое объяснение.
```

## Правила

- Применяй `criteria` и `interests` только в режиме анализа видео.
- Всегда отвечай только на русском языке.
- Возвращай только итоговый ответ без reasoning и служебных метаданных.
- Не давай поверхностный пересказ; выделяй смысл, факты и практические выводы.
- Ограничивай итог примерно до 3200 символов, чтобы ответ помещался в Telegram.
- Если transcript недоступен, верни понятную ошибку с причиной.
- Не сохраняй transcript и результаты на диск без явной необходимости.
