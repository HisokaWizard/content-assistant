# Spec 02: Настройка Telegram Бота

## Цель

Создать Telegram-бота как UI для взаимодействия с opencode агентом.

## Шаги

### 2.1 Создание Telegram Бота

- Открыть BotFather в Telegram (`@BotFather`)
- Создать нового бота командой `/newbot`
- Задать имя бота и username
- Получить API token
- Сохранить токен в `.env` файле (тема для безопасности — не коммитить в репозиторий)

### 2.2 Базовые Команды Бота

Реализовать следующие команды:

- `/start` — приветствие, краткое объяснение возможностей
- `/help` — показать все доступные команды
- `/clear` — очистить сессию пользователя

### 2.3 Команда Анализа YouTube

- `/analyze <youtube_url>` — явно анализировать видео
- Также автоматически обнаруживать YouTube URL в любом сообщении
- Показывать индикатор "анализирую..." во время обработки

### 2.4 Настройка Интересов и Критериев

- `/interests <text>` — сохранить интересы пользователя
- `/criteria <text>` — сохранить критерии оценки видео

### 2.5 Интеграция с Opencode Agent API

- Настроить HTTP клиент для отправки запросов к opencode агенту
- Обрабатывать ответы и форматировать для Telegram
- Обрабатывать ошибки и показывать пользователю

### 2.6 Обработка Потока Сообщений

- Реализовать эхо-ответ для обычных текстовых сообщений (без YouTube URL)
- Показывать typing indicator во время генерации ответа

## Критерии Приемки

- [ ] Бот создан и token получен
- [ ] Команды `/start`, `/help`, `/clear` работают
- [ ] `/analyze <url>` запускает анализ
- [ ] YouTube URL автоматически обнаруживается
- [ ] `/interests` и `/criteria` сохраняются
- [ ] Результат анализа отображается в читаемом формате

## Зависимости

- Telegram Bot API (библиотека: grammY или node-telegram-bot-api)
- Доступ к opencode agent API
