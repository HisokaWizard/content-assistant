#Spec 01: Настройка Opencode Agent

## Цель

Настроить opencode в режиме сервера с агентом для обработки YouTube-видео.

## Шаги

### 1.1 Проверка установки opencode

```bash
which opencode
opencode --version
```

### 1.2 Инициализация агента

- Создать директорию для агента (например, `agent-youtube-analyzer/`)
- Настроить базовую конфигурацию агента
- Определить системный промпт с ролью: анализ YouTube-видео, генерация summary, работа с сессией пользователя

### 1.3 Реализация Tool: YouTube Transcript

- Найти и настроить способ получения transcript (youtube-transcript-cli или аналог на Node.js)
- Реализовать функцию получения транскрипта по YouTube URL
- Обработать ошибки: видео без transcript, недоступное видео

### 1.4 Реализация Tool: Translation

- Определить способ перевода (использовать LLM с функцией перевода)
- Реализовать перевод transcript на русский язык

### 1.5 Реализация Tool: Summary Generation

- Использовать LLM для генерации summary + тезисов + рекомендации
- Использовать критерии пользователя для оценки
- Форматировать результат структурированно

### 1.6 Настройка Server Mode

- Запустить opencode agent в режиме сервера
- Настроить endpoint для приема запросов
- Проверить работоспособность через curl/http запрос

## Критерии Приемки

- [ ] opencode работает в режиме сервера
- [ ] По YouTube URL возвращается transcript (или ошибка, если недоступен)
- [ ] Transcript переводится на русский
- [ ] Генерируется summary, тезисы (3-5), рекомендация (смотреть/не смотреть)
- [ ] Состояние пользователя сохраняется между запросами

## Зависимости

- Node.js (youtube-transcript-cli или аналог)
- Доступ к LLM (через opencode)
